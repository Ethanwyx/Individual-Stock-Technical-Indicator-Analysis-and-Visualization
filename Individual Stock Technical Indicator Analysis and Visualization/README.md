# 个股技术指标分析与可视化

## 项目简介
本项目是一个基于Python的个股技术指标分析与可视化工具，能够获取A股股票数据、计算常用技术指标并进行交互式可视化展示。

## 功能特点

### 数据获取
- 支持获取任意A股股票的历史行情数据
- 支持自定义时间范围
- 数据自动保存到本地，避免重复请求

### 技术指标计算
- **移动平均线(MA)**：支持5日、10日、20日、60日均线
- **MACD指标**：包含MACD线、信号线和柱状图streamlit run app.py
- **KDJ指标**：包含K线、D线和J线
- **RSI指标**：相对强弱指标
- **布林带(BOLL)**：包含上轨、中轨和下轨
- **成交量指标**：包含成交量柱状图和OBV指标

### 可视化展示
- 交互式K线图与均线叠加
- 各指标独立图表展示
- 组合图表，包含所有指标的综合展示
- 支持缩放、悬停显示数据等交互操作
- 响应式设计，适配不同屏幕尺寸

### 用户界面
- 简洁直观的Web界面
- 股票代码输入框
- 时间范围选择器
- 指标选择复选框
- 数据表格查看
- 统计信息展示

## 技术栈

- **数据获取**：akshare
- **数据处理**：numpy、pandas
- **技术指标计算**：TA-Lib
- **可视化**：plotly、matplotlib
- **Web框架**：streamlit

## 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

### 3. 访问应用

在浏览器中访问命令输出中显示的URL，默认为：http://localhost:8501

## 使用说明

1. 在侧边栏输入股票代码，如：600000
2. 选择时间范围
3. 勾选要查看的技术指标
4. 点击"开始分析"按钮
5. 在主界面查看图表和数据

## 注意事项

1. 首次运行时，需要从网络获取数据，可能需要一些时间
2. 确保网络连接正常，尤其是首次运行时
3. 部分技术指标需要一定的数据量才能计算，建议选择较长的时间范围
4. 本工具仅用于学习和研究，不构成投资建议
